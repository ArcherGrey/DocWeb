:question: `markdown`：

- **markdown 的锚点跳转标签必须是英文小写(全部都要)**
- **锚点多个单词之间可以使用`-`连接**

技术相关：

- `vue`
- `element` (配合 `vue` 的 `ui` 框架)
- `marked.js` (解析 `markdown` 文件)
- `anime` (动画库)
- `webpack` (打包工具)

前端常用变量（文档链接）保存在 `json` 中：

- `mainmenu.json` 保存中间跑马灯图片链接
- `sidemenu.json` 保存侧边选中后的文档链接，添加新的文档就在这个文件中添加，通过 `json` 可视化编辑器添加很方便
